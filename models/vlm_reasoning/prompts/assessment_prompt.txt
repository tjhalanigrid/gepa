You are given an image of a damaged vehicle.
Perform a thorough visual inspection to identify and analyze damages.

1. Identify all visible damages (dents, scratches, cracks, tears, punctures, smashed components, etc.)
2. Identify all affected vehicle parts (bumper, fender, door panel, hood, headlight, windshield, wheel, etc.)
3. Estimate damage severity (Pristine, Minor, Moderate, Severe) for each identified damage with justification.
4. Provide a confidence score (0.0 to 1.0) for each detection.
5. Provide an overall claim summary and estimated severity.

Format Rules:
- Return ONLY a valid JSON object. No pre-amble, no post-script, no explanations outside the JSON structure.
- Do not hallucinate or guess details that are not visible.
- Reference the image filename for every damage location to ensure auditable evidence.

Return your analysis strictly in the following JSON schema:
{
  "claim_id": "string",
  "overall_summary": "string",
  "overall_severity": "Pristine | Minor | Moderate | Severe",
  "damages": [
    {
      "part": "string (e.g., front bumper, left door panel, right headlight)",
      "damage_type": "string (e.g., dent, scratch, cracked, shattered)",
      "severity": "Pristine | Minor | Moderate | Severe",
      "supporting_images": ["string (image filename)"],
      "confidence": 0.00,
      "reasoning": "string justifying the assessment based on visual evidence"
    }
  ]
}
